Adding Latex Package

January 29, 2023

To add a latex package like tikz-timing.

  1. search it on CTAN
  2. Download the zip
  1. extract it in $TEXMFHOME/tex/latex/<packagename>/ I have export TEXMFHOME=$LOCAL_USR/share/texmf/ in the environment.
  2. cd to that directory
  3. Run pdflatex <packagename>.ins
  4. Run texhash

Now the package should be available to pdflatex.

If you are using VSCode with “LaTeX Workshop” extension then you have to tell it about the environment variable $TEXMFHOME.

Similarly to what is explained here.

  1. go to extension settings
  2. search tools
  3. edit json
  4. add the ’env’ variable to match your $TEXMFHOME variable

The reason you have to use the absolute path is that the extension can’t expand the variable as written in it’s wiki
