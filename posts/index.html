Adding Latex Package

2023-01-29 ::
To add a latex package like tikz-timing. search it on CTAN Download the zip extract it in $TEXMFHOME/tex/latex/<packagename>/ I have export TEXMFHOME=$LOCAL_USR/share/texmf/ in the environment. cd to that directory Run pdflatex <packagename>.ins Run texhash Now the package should be available to pdflatex. If you are using VSCode with “LaTeX Workshop” extension then you have to tell it about the environment variable $TEXMFHOME. Similarly to what is explained here. go to extension settings search tools edit json add the ’env’ variable to match your $TEXMFHOME variable The reason you have to use the absolute path is that the extension can’t expand the variable as written in it’s wiki
